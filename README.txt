This is a Simple PHP Sign up & Login System (you can login with email or username)- After you have logged in as a user you can find a dashboard where you can chat with other users or link a photo that will immediately be displayed in the page .
- After you have logged in as an Admin you can do the same tasks as the user with an extra section that drives you to the admin dashboard , When you are in the Admin dashboard you can have access to all users infos including yourself and can delete any wanted User .
Have Used 2 separate databases one for chat and users infos and the other one is for photos , the photos are displaced to a file where they are stored and ready to be displayed . (This can be improved and use only one database)
